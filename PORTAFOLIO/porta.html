Estudios

  • Consulta 1:
    • Protocolo web no cifrado que usa el puerto 80, permite la transferencia de datos en la web.
    • TCP/IP es un protocolo de la capa de transporte que permite la transferencia de datos en la red.
    • DNS es un protocolo que permite la traducción de nombre a IP.
    • El CDN es una red de servidores que almacenan copias de contenido web, para entregarlos más rápido a los usuarios.
    • El WWW se refiere a la World Wide Web, que permite la conexión mundial a través de hipervínculos.
    • El FTP se usa para la transferencia de archivos, usando el puerto 20 y 21.
    • El HTML es un lenguaje que se usa principalmente para la creación de páginas web.
    • El DOM es una representación programática de una página web que permite a los desarrolladores interactuar con su contenido usando código.
